Token marketing
===

## Setup

Building the project requires `yarn`.
[Install yarn](https://yarnpkg.com/en/docs/install), then the necessary dependencies:
```
$ yarn
```

## Development

```
npm run dev
```

Runs a local server on [localhost:3000](http://localhost:3000).

## Production

```
npm run prod
```

## Deployment

```
npm run gh-pages
```
